# SpendWise Finance Platform

## Problem Statement

SpendWise is a minimal finance platform concept for a bank-backed service that helps users understand their spending patterns and suggests investment tips based on those patterns. The goal is to create a simple, responsive website that highlights semantic HTML, modern layout techniques, and a tiny amount of JavaScript for finance-related data.

## Features

- Semantic page structure using `header`, `nav`, `main`, `section`, `article`, `aside`, and `footer`
- Responsive layout with CSS Grid, Flexbox, and relative sizing
- Tailwind CSS via CDN for fast styling and consistent design
- Anchor navigation using IDs for same-page section jumps
- Clean UI with dark finance-inspired palette and spacing
- Minimal JavaScript for a domain-related market insight example

## Technologies

- HTML5
- CSS3
- JavaScript (vanilla)
- Tailwind CSS CDN
- Coindesk public API for live market data example

## Limitations

- No backend or real user authentication
- Only a single API call is used for demo purposes
- Spending analysis is presented with static example values
- Not a full trading platform, only a concept landing page

## Future Scope

- Add real spending analytics from bank transaction data
- Build charts and dynamic user dashboards
- Create login flows and personalization
- Expand investment tips with machine learning or budget rules
- Support multiple currencies and risk profiles

## AI Tools Used and Purpose

- GitHub Copilot / AI assistant for generating the website structure and documentation quickly.
- Tailwind CSS CDN to simplify design without writing a large stylesheet.

## Learning Reflections

This assignment reinforced how semantic HTML improves accessibility and page structure. I practiced responsive UI using grid and flex layouts, and I kept JavaScript deliberately small while still integrating a real finance data source. The result is a simple, modern landing page that can be extended later into a richer banking or investment experience.
